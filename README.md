# awale-symfony

Experimental Awale bot on Slack

## Run Webserver

```
make run
```

Your web server is up and running at [http://localhost](http://localhost)

## Stop Webserver

```
make stop
```

## Expose Webserver

```
make expose
```

## Update Submodule

```
git submodule update --recursive --remote
```

## Test

```
make test
```
